Indigo accent (ember)

The brand color. Use ONE primary CTA per view; the rest stay neutral. Tonal/outline are good for secondary accent actions.

Neutral (forge)

Cancel, secondary actions, and the inline “New Project / New Board” add-pills. These carry the bulk of the UI.

Destructive (red)

Delete board, sign out, remove. Reserve solid red for irreversible confirms; tonal for in-card delete affordances.

Status accents

Green for confirm/success; amber (--fm-wait) for the agent “waiting for input” state. Both are theme-aware.

Sizes

Same tokens, two scales. sm for dense rows and section headers, default for modals and primary CTAs.
